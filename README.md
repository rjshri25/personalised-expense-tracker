# 📊 Personalised Expense Tracker

A full-stack web application to track daily expenses, manage monthly budgets, and visualize spending patterns. It helps users gain better control over their finances through an intuitive dashboard and categorized expense tracking.

---

## 🚀 Live Demo

👉 [https://personalisedexpensetracker.netlify.app](https://personalisedexpensetracker.netlify.app)

---

## 🧠 Features

* 💰 Add daily expenses with category & date
* 📊 Visual dashboard for spending insights
* 📅 Monthly expense tracking
* 🧾 Income vs expense comparison
* 🔍 Category-wise analysis (Food, Travel, etc.)
* 📱 Responsive UI for mobile & desktop
* ⚡ Real-time updates

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* HTML5, CSS3
* JavaScript (ES6+)
* Chart.js (for graphs)

### Backend:

* Node.js
* Express.js
* MongoDB (Atlas)
* Mongoose

### Deployment:

* Frontend: Netlify
* Backend: Render
* Database: MongoDB Atlas
